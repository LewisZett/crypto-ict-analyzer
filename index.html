Crypto ICT Price Action Analysis App

Screens the CoinGecko top 50 by market cap, filters for momentum + local ICT-style confluence (moving average, market structure, Fibonacci OTE) computed in JavaScript, then asks Gemini AI to judge context/session quality only for candidates that already pass the hard technical rules.

Disclaimer: Educational/research tool only. Not financial advice. AI-generated trade plans must be manually verified before acting on them.
Security: Your Gemini API key is sent directly from your browser and is never stored. Use only on a trusted device.
Settings (risk, thresholds, timezone)
Backtest (no API key needed) - tests the local filter logic against recent historical data

This does NOT call Gemini - it only replays the deterministic MA/structure/OTE/SL/TP rules against past candles, so you can see how the filter itself would have performed before trusting any of its live signals.

Trade Journal - every recommendation from a live run is saved here automatically

Nothing here is graded until you click "Check Outcomes" - that pulls the real candles that happened after each trade was logged and checks whether price actually hit TP or SL. This is the honest record of how live signals did; the cards above are not.
